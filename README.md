# Sentiment Analysis Project

This project uses a pre-trained DistilBERT model to analyze the sentiment of text.

## 🚀 Features
- AI-based sentiment analysis
- Uses Hugging Face Transformers
- Simple and easy to use
- Command-line interface

## 🧠 Model Used
DistilBERT (distilbert-base-uncased-finetuned-sst-2-english)

## 📦 Requirements
- Python 3.x
- transformers
- torch

## ▶️ How to Run
1. Install dependencies:
   pip install transformers torch

2. Run the project:
   python app.py

## 💡 Example

Input:
I love this project

Output:
Sentiment: POSITIVE  
Confidence: 99%
